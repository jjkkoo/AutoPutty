Welcome to AutoPutty.

AutoPutty is a tabbed and dockable PuTTY session manager written in Qt:

$ git clone https://github.com/jwhitmore/AutoPutty.git AutoPutty
$ cd AutoPutty
$ git checkout origin
$ qmake
$ make

Origin

AutoPutty started out because of one reason, to make PuTTY more efficient with multiple *nix machines. Original intent was to be small and just auto-login to your sessions, but developed into a pretty decent way of handling PuTTY connections/sessions. There are other connection managers for PuTTY that also use tabs and can auto log you in, but they didn't layout the way I wanted so I decided to write my own. Enjoy.
